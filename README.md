# Finance Management System Simulator

A lightweight, front-end simulation of a personal finance management dashboard.  
This project is built as a static demo to visualise income, expenses, and basic financial summaries – ideal for testing UI concepts or learning how financial data can be presented.

> **Note:** This is a **simulator** only. No real financial transactions are processed; all data is mock and client-side.

## Features

- **Dashboard overview** – quick glance at total balance, income, and expenses.
- **Add transactions** – simulate adding income or expense entries (UI only).
- **Transaction list** – view and filter mock transaction history.
- **Responsive layout** – works on desktop and mobile screens.

---

## Technologies Used

- **HTML5** – structure  
- **CSS3** – styling (custom, no frameworks)  
- **JavaScript (ES6)** – interactivity and mock data handling  

(No server-side logic or database – runs entirely in the browser.)


1. Place the project folder inside your web server’s root directory (e.g., `C:\xampp\htdocs\FinanceGo`).
2. Start Apache (or your chosen server).
3. Open your browser and navigate to: http://localhost/FinanceGo/index.html
